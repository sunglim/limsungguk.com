이런 형태의 데이터가 있다면 세타0, 세타1은 이렇게 그려야할것입니다 그러면 어떻게 이들을 구할것인가? training example에서 h세타(x)가 y값에 가까운 값을 세타0,세타1을 고르면된다. Formalize 해봅시다. Linear regresion에서 우리가 해야할것은 세타0, 세타1을 최소화하는것이므로 (h세타(x) - y)^2이 최소화되는것이다. SUM ((h세타(x) - y)^2) (1부터 m까지) ;x는 square of house, y는 actual price of house. cost function J는 1/2m SUM ((h세타(x) - y)^2) (1부터 m까지) . 이것은 세타0, 세타1을 최소화하는것이므로 J(세타0,세타1) 으로 나타낼수있다. = cost function. Square cost function이라고 부르기도한다. 다음비디오에서 cost function J가 무엇을 하는지 더 자세히 알아봅시다.
