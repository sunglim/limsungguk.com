Logistic Regression

Classification

Classification problem에 대해 알아볼것입니다. logistic regression이라는 알고리즘을 개발할것인데, 요즘날에 가장 널리 쓰이는 알고리즘 입니다. 이메일이 스팸인지 아닌지, 혹은 암이 악성인지 양성인지를 구분하는것입니다. y <- {0,1}로 나타낼수 있는데 0: 'Negative class', 1: 'Positive class'로 나타낼수있다. 구분하는것일 뿐이고 0,1. negative positive가 갖는 의미는 없다. 구분을 여러게로 하는경우 multi classficiation problem이라고 하고 두게이면 binary classification problem이다. 예제 그림, hypothesis함수가 0.5를 기준으로 0,1을 구분하게된다. 하지만 극단적으로 Tumor size가 매우 큰값이 추가된다면 hypothesis의 직선의 기울기가 바뀌게 되고 이것은 좋지않다. 따라서 linear regression의 hypothesis함수를 classification problem에 사용하는것은 일반적으로 좋지않다. classification problem에서 결과는 0,1이 나오지만 linear regression에서는 h(x) > 1, h(x) < 0인경우가 발생하기에 적절하지못하다. 따라서 logistic regression; 0<= h(x) <= 1을 이용할것이다.
